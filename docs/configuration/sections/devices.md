# Configuration file - SCSmartDevices section

In this section you can configure one or more Shelly and/or Tasmota smart switches, one of which will be used to control or meter your electrical device. See this page for an example yaml file configuration:

[Example Smart Device Configuration](https://nickelseyspelloc.github.io/sc-smart-device/example_config/)


## More information

See the [Smart Device Getting Started guide](https://nickelseyspelloc.github.io/sc-smart-device/) for details on how to configure this section.
