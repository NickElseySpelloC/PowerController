# Upgrading the PowerController app

Occasionally, or if you're having problems, you can try upgrading to the latest version of the PowerController app

```bash
cd /home/nick/scripts/PowerController       # cd into the project root folder

git reset --hard origin/main                # Reset to the remote main branch 
```
